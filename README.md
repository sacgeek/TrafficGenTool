# TrafficGenTool
 Network Lab Traffic Generation Tool
This tool is intended to be used ONLY in a lab environment with limited scale to simulate user traffic that looks like MS Teams traffic (only between worker nodes) and/or a small amount of web and/or youtube traffic.
As a lab tool there is no security. All traffic is sent in plain text and there is no validation of certificates for websites downloaded by the tool.
The tool can be used in a completely isolated lab with no access to the internet but will then be unable to run tests that download from internet based websites or from YouTube. 
I built this with the help of Claude AI for use in my own network lab and share freely as-is with no warranty or guarantee whatsoever.
